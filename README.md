# InvesTreeTest

- [NodeJS](https://nodejs.org)
- [NPM](https://npmjs.com) / [Yarn](https://yarnpkg.com)
- [Angular-cli](https://www.npmjs.com/package/@angular/cli)

# Installation
- clone atau download source
- Run npm i atau yarn install
